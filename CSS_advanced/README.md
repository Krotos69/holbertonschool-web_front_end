# 📘 CSS Concepts Project  
A structured overview of essential CSS concepts, including selectors, layout behavior, resets, variables, animations, transforms, and more. This project serves as a learning reference and practical guide for mastering foundational and intermediate CSS techniques.

---

## 📌 Project Overview  
This project compiles key CSS topics into a single, organized resource. It explains how CSS works under the hood, how to write maintainable styles, and how to apply modern techniques such as variables, gradients, animations, and transforms. It also highlights differences between styling methods and icon systems, helping developers make informed decisions in real projects.

## 🧠 Concepts Implemented  

### ✔ Selectors, Properties & Values  
Demonstrates how CSS targets elements and applies styling rules.

### ✔ Block vs Inline Elements  
Explains layout behavior differences and how they affect spacing and structure.

### ✔ CSS Reset  
Ensures consistent styling across browsers using a universal reset or Normalize.css.

### ✔ CSS Variables  
Shows how to define and reuse custom properties for scalable design systems.

### ✔ Inline vs Embedded vs External CSS  
Compares the three main ways to apply CSS and when to use each.

### ✔ Float‑based Grid Systems  
Illustrates how traditional grid layouts were built before Flexbox and Grid.

### ✔ Icon Webfonts vs SVG Icons  
Covers pros and cons of each method and modern best practices.

### ✔ Pseudo‑classes vs Pseudo‑elements  
Clarifies the difference between element states and element sub‑parts.

### ✔ Background Gradients  
Includes examples of linear, radial, and conic gradients.

### ✔ CSS Animations  
Uses `@keyframes` and animation properties to create motion.

### ✔ 2D & 3D Transforms  
Demonstrates rotation, scaling, translation, and perspective.

### ✔ Vendor Prefixes  
Explains browser‑specific prefixes and how tools like Autoprefixer help.

## 🎛️ Variables Used  
Defined in `variables.css`:

```css
:root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
  --text-color: #333;
  --spacing: 1rem;
  --radius: 6px;
}
```

These variables are reused across components for consistent theming.
